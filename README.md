# 🧮 Calculadora de IMC em Python

Projeto desenvolvido como prática de lógica de programação em Python.

## 📌 Sobre

O programa calcula o IMC (Índice de Massa Corporal) com base no peso e altura informados pelo usuário e exibe a classificação de acordo com a tabela da OMS.

## 🛠️ Tecnologias

* Python 3

## ▶️ Como executar

1. Clone o repositório:

```
git clone https://github.com/Ribeiroqz/Calculadora-de-IMC-Python-.git
```

2. Acesse a pasta:

```
cd Calculadora-de-IMC-Python-
```

3. Execute o arquivo:

```
python calculadora_imc_py
```
